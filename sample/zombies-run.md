# Zombies, Run! — Visual Design Research

## Overview

Zombies, Run! (Six to Start) is the deepest narrative-as-habit-engine in the market: 500+ missions across 13+ seasons cast the runner as "Runner 5," a courier for post-apocalyptic Abel Township. Each run plays a chapter of audio drama interleaved with the user's own music, and the runner collects supplies (medkits, sports bras, batteries) that feed a persistent isometric base-building meta-loop. The "supplies → township" mechanic is functionally identical to Tend's "offerings → deity" loop: a daily ritual act produces narrative + tangible accumulation in a world you are quietly rebuilding.

## Design tone: post-apocalyptic gritty

The visual language: charcoal/black backgrounds, blood-red accents, weathered serif/slab display type, hand-illustrated isometric township, photographic character portraits with desaturation. It is unmistakably horror-adjacent without being lurid — restrained enough to wear in public, distinctive enough to be a lifestyle marker.

---

## Design language & takeaways for Tend

- **Narrative is the habit-loop's reward, not its garnish.** ZR plays a chapter only when you run — the story literally does not advance without the behavior. Tend's deity dialogue/lore should be gated the same way: offerings unlock the next beat.
- **Audio-primary UIs need almost no UI.** During a run, three big numerals + a now-playing strip is enough. For Tend's ritual/meditation moments, resist filling the screen — let voice/ambient carry, keep visuals glanceable.
- **The base/township IS the home screen.** Accumulation is visible the moment the app opens; there is no separate "progress" tab. Tend's altar should be the home view — every habit completed should visibly add to it.
- **Each unit of accumulation deserves its own illustration.** Buildings, supplies, equipment all have distinct art. Tend's offerings, altar items, and deity favor states should be illustrated, not iconified — accumulation is only motivating if it's beautiful.
- **Scripted interval triggers (the "chase") create stakes without gamification chrome.** A red alert + audio cue does what XP bars try to do. Tend can use scripted ritual "tests" or seasonal events as narrative-driven intensity spikes.
- **Ship narrative as seasons, not as content drops.** ZR's 13+ seasons give long-term users something to anticipate without new mechanics. Tend's deities/pantheons can release as seasonal arcs — same engine, refreshed lore.
