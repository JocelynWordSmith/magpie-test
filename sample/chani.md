# CHANI — Visual Design Research

CHANI (by astrologer Chani Nicholas) is the gold standard for the aesthetic midpoint Tend is targeting: **warm enough to feel like self-care, dark enough to feel witchy**. It pairs cut-paper collage art, vintage botanical illustration, magazine-quality long-form writing, and an explicitly queer, gender-expansive voice. No glossy gradients, no edgy goth metal, no generic "moon phase" wellness clip-art — CHANI treats astrology as both ritual and literature, and the UI reads like a beautifully art-directed indie zine that happens to run on your phone.

For Tend — where habits are "offerings" to patron deities — CHANI is the closest production reference for: how to make a daily ritual feel sacred without feeling cringe, how to use dark backgrounds without going emo, how to write copy that is gentle but never twee, and how to design inclusion as an aesthetic statement rather than a checkbox.

## Tone & Inclusivity

This is what most "witchy" apps get wrong and what CHANI gets uniquely right — and it's directly relevant to Tend's "offerings to patron deities" framing, which can read either as devotional or as cringe-y depending on tone.

- **Queer and gender-expansive by default, not as an option buried in settings.** Pronouns are surfaced at onboarding. Sign descriptions avoid gendered partner assumptions. Compatibility / synastry doesn't assume a man-and-woman pairing. This is presented as the *normal* product, not a "diversity mode."
- **Therapist-adjacent language, not influencer-wellness language.** Copy talks about boundaries, nervous system, grief, repair, ancestors. It does not say "manifest," "high-vibe," or "your best self." Tend should adopt this register — devotional but grounded.
- **Astrology framed as agency, not fate.** Horoscopes describe what's possible, not what will happen. Patron-deity framing in Tend should follow this: deities as relational guides you tend, not bosses who judge.
- **Politically aware.** CHANI is openly anti-capitalist, pro-Palestine, pro-trans rights — the brand has values and shows them. This grants permission for warmth without saccharine neutrality.
- **Inclusivity expressed visually, not just verbally.** The collage art uses Black, brown, fat, queer, disabled bodies as default — not "diverse" as a campaign but as the everyday source material. Tend's deity art direction should follow.
- **Gentle without being childish.** The tone is "wise older queer friend who has done the work" — not "fairy princess" and not "edgy crone." This is the exact register Tend wants.

## Design language & takeaways for Tend

1. **Collage > illustration > 3D render > photo.** Hand-cut collage from vintage botanical / anatomical / occult sources is CHANI's single most distinctive move. It costs more than icon packs but is the entire reason the app feels human. Tend should commission a small set of collage illustrations per patron deity rather than ship generic icons.
2. **Warm-dark palette: oxblood, terracotta, butter, cream, midnight blue — never pure black.** True black reads goth-edgy and clinical. CHANI's "dark" is always a deep blue or oxblood with warmth. Tend's witchy-but-cozy target should adopt this palette as the floor, not pure #000.
3. **Magazine typography pairing.** A heavy editorial serif for display (think Tiempos / Canela / GT Sectra) + a humanist sans for body (think GT America / Söhne). Tend should resist the "rounded geometric sans" wellness default — it reads like every other wellness app.
4. **Long-form is a feature, not a wall of text.** CHANI's weekly horoscope essay is treated as the prestige content. Long line-length, generous leading, drop caps, real paragraph breaks. Tend's "offering descriptions" and deity lore could be a weekly long-form essay in this exact register — a reason to open the app beyond ticking habits.
5. **Affirmations as designed objects, not floating text.** Each affirmation gets full-bleed color + serif + collage cutout, share-ready. Tend's "offering complete" celebration should be one of these — a screenshot-worthy designed card per completion, not a green checkmark.
6. **Gender-expansive astrology as design statement.** Pronouns + identity surfaced in onboarding, copy that never assumes gender of self or partner. For Tend: patron-deity selection should mirror this — let the user define their relationship to the deity (devotee, sibling, student, kin) instead of defaulting to gendered hierarchy.
7. **Ritual framing trumps habit framing.** CHANI's daily content is called a "ritual" not a "task." It has a beginning (intention), middle (breath / journal / read), end (affirmation card). Tend's "offerings" should mirror this structure — every habit completion is a small three-beat liturgy, not a tap.
8. **Editorial grid, modular cards.** Today screen = stacked cards each with collage hero + serif title + sans deck. Tap to expand inline rather than route to new screen. This pattern preserves the magazine-spread feel.
9. **No urgency, no streaks-as-shame, no red.** CHANI never uses red urgency, countdowns, "you missed a day" guilt. Tend should resist the habit-app habit of shame-as-engagement; offerings missed should be re-offered, not penalized.
10. **Pricing as invitation, not gate.** CHANI Plus paywall reads as "deeper practice available" — not "unlock features." For Tend's eventual monetization, frame the upsell as access to richer deity lore / longer essays / more collage art rather than feature gating.
11. **The print planner exists.** CHANI's physical planner uses the same visual system. This is a strong hint that Tend's visual language should survive off-screen — if it can't be a real journal page, it's probably too glossy.
12. **Politics + values are part of the aesthetic.** CHANI is not neutral and is more trusted for it. Tend's "patron deity" framing already commits to a worldview; lean into that with the same confidence rather than hedging toward generic wellness.
