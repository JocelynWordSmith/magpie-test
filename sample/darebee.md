# Darebee — Visual Design Research

Darebee is a free, donation-funded, ad-free fitness library running narrative RPG-style bodyweight programs. Each program reads like a graphic novel: comic-book story panels deliver chapter-by-chapter fiction, while signature clean infographic diagrams instruct the body. Hero's Journey — a 60-day branching narrative workout — is the flagship. The library spans Superhero, Sci-Fi (Star Trek), Fantasy/RPG (knights, mages, witchers), Ninja, Zodiac, and seasonal programs. For Tend, the infographic + comic-panel formula is a direct model for **ritual instruction UI**, and the themed-collection model maps cleanly to **sabbat-themed offering programs** (Samhain, Yule, Beltane) with patron-deity archetypes (Mage, Valkyrie, Druid).

## Color palette

- **Heroic primaries:** comic-book red/blue/yellow (Superhero collection)
- **Dark fantasy:** deep violet, blood crimson, charcoal (Shadebound, Dark Elf, Scorpio) — most Tend-relevant
- **Mystical:** arcane purple + gold (Mage, Paladin) — patron-deity palette territory
- **Nature/pagan:** forest greens + bone whites (Druid) — Beltane/Litha territory
- **Sci-fi:** electric blues, alert reds, industrial gray
- **Unifier:** off-white/cream page background with heavy black ink — every program shares this base, theming is achieved via accent color only

## Typography

- Bold condensed sans-serif logotypes per program (often custom display faces)
- All-caps short labels on infoboxes and exercise names
- Large numeric rep counts in heavy weight — the most prominent type on every card
- Body narrative set in clean serif or humanist sans — book-like, not UI-like

## Illustration style consistency

A single illustrator-driven house style: hand-drawn ink linework, flat color fills, minimal gradient, character poses rotated through a reused figure library. Theming = palette swap + costume swap on a shared chassis. This is the operational lesson: one template, infinite themes.

---

## Design language & takeaways for Tend

- **Infographic ritual cards.** Adopt Darebee's exercise-grid template for daily offerings: deity sigil + accent color in the title bar, ritual steps in a numbered grid, "rep counts" replaced with offering counts (3 candles, 7 breaths, 1 chant).
- **Comic-panel story chapters.** Pair each ritual card with a short illustrated narrative beat — the patron deity's ongoing story unfolds as the user completes offerings, turning habits into serialized myth.
- **Narrative branching as habit mechanic.** Steal Hero's Journey's choice nodes: at sabbat thresholds the user picks a path (Shadow / Light, Crone / Maiden) that forks the next month's offering set and deity-relationship arc.
- **Themed program collections = sabbats + deity archetypes.** Mirror the Zodiac twelve-pack with eight sabbat programs and a deity-archetype roster (Mage, Druid, Valkyrie, Warrior Queen already exist as visual proofs of concept).
- **One chassis, infinite themes.** Build one ritual-card component; theme it via palette + sigil + typeface swap per deity. Darebee proves this scales to 100+ programs without redesign.
- **Donation-funded, ad-free ethos.** Darebee's no-ads, no-paywall, donation-supported model signals trust and craft — a strong positioning anchor for Tend against extractive habit-app norms.
